ClearVision

ClearVision is an automatically updating, easily customizable theme for Discord.

ClearVision is built for use with BetterDiscord.

For update notifications and theme support, join the ClearVision Discord server.

If you would like to contribute, check out the Github Repository.

ClearVision v5

This is the public Theme version of ClearVision for BetterDiscord.

If you wanna download ClearVision v5, head over to the Github Releases and download the Theme of your choosing. You can find installation instructions in the ClearVision Wiki.

ClearVision v6

This project is still work-in-progress currently. The goal is to do a clean, easy-to-update rewrite of the ClearVision themes.

Participation in the development process is currently limited to Contributors, Donators and Testers.

However, there is a public BETA preview currently available on the the ClearVision Discord server.

Installation Instructions
1. Installing BetterDiscord

The installation is different depending on which operating system you are on.

Visit BetterDiscord.net and read the BetterDiscord Wiki for installation instructions.

If you have any questions, join the BetterDiscord Discord server.

2. Downloading the Theme

Got to the Latest Github release and download the theme of your choosing.

3. Installing the Theme

Move the theme file to your BetterDiscord theme folder:

Windows: %appdata%\betterdiscord\themes\ Mac: ~/Library/Preferences/BetterDiscord/themes

If you have no idea how to open the according directory:

Open Discord
Open your user settings
Go to the BetterDiscord tab
Go to the themes tab
Click on the Open Theme Folder button
4. Enabling the theme

Make sure the theme file is placed in the correct folder and the file extension is .theme.css
(If you dont have "Show file extensions" enabled it will only show .theme)

Now head back to Discord and restart it.
(Instead of restarting the whole programm you can reload using Ctrl + R)

Head to the themes tab and check the checkbox next to the theme you just installed!
